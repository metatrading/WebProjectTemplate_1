# Spring Boot + Thymeleaf + JDBI + TestNG + HTML Unit  
このプロジェクトは、以下のJava Webアプリケーションの構成テンプレートを提供します。

## 採用フレームワーク

| レイヤ           | フレームワーク名 |
|:-----------------|:-----------------|
| Webサーバ        | Spring Boot      |
| HTMLテンプレート | Thymeleaf        |
| DBAccess         | JDBI             |
| Test             | TestNG           |
| Web Test         | HTML Unit        |

## このプロジェクトテンプレートに用意したもの
* 各フレームワークの利用サンプルコード
